<h2>Code Installation: </h2>
<ul>
	
<li>

</li>
<li>
Downlod the pickle file of the model from this <a href = "https://drive.google.com/open?id=1M4X7ccsQCNciqAJ3SCGIDHtZLsvJEni6">Link</a>

<pre><code>pip install tensorflow
